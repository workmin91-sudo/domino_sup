# Vercel API 토큰 발급 방법

## 토큰 발급 단계

1. **Vercel 대시보드 접속**
   - https://vercel.com/account/tokens

2. **토큰 생성**
   - "Create Token" 버튼 클릭
   - 토큰 이름 입력 (예: `auto-deploy-token`)
   - "Create" 클릭

3. **토큰 복사**
   - ⚠️ **중요**: 토큰은 한 번만 표시됩니다!
   - 토큰을 복사하여 안전한 곳에 보관

4. **토큰 제공**
   - 토큰을 알려주시면 자동 배포를 진행하겠습니다

## 보안 주의사항

- 토큰은 비밀번호처럼 취급하세요
- 공개된 곳에 공유하지 마세요
- 필요시 언제든지 토큰을 삭제할 수 있습니다
